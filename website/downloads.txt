====== Downloads ======

==== our $VERSION=0.28 ====

**This program depends on a number of perl modules including the Glib and Gtk2 perl bindings.** Most linux distributions have standard packages for these bindings, if your distribution has these packages installing them will probably be much easier then compiling them manually or getting them from CPAN.

In order to compile and install the source run:

'''
$ perl Build.PL
$ ./Build test
$ ./Build install
'''

More detailed instructions to build from source can be found [[:Install|here]].
